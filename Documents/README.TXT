Description:
	Base Analyzer Tool is a software for generating codon usage table with RSCU value comparing with base sequence, GC count in different position and rich region of a sequence in fasta format.

Main Features:
    * Generate codon usage table for multiple sequence comparing with base sequence and without comparicng;
    * Generate GC count in different position and in whole sequence;
    * Generate AT and GC Rich region;

Installation:
	1. install ubuntu 11.10 or upper version
	2. install python 2.7 or upper version
	3. install PyQt 4 or upper version
	4. download the software from website.unzip it any place of your copputer.
	5. make the genePatternAnalyzer.pyw file executable using the command chmod +x
